MuttHead

MuttHead is a simple programming language. It is the successor to "LaLa", my first "language". It wasn't really a programming language, and it was very buggy. It didn't implement any of the techniques that real programming languages use, so I decided to remake it. Originally I was going to call my new language "BlaBla", but I like MuttHead better (shout out to Cocoa).
