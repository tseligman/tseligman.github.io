Translatable Gibberish

About
For any word with L number of characters and V number of vowels, it can have up to (L-V-2)!* different spellings!

Some examples:

  • Garbage has 2 possible spellings in TG
  • Scrambled has 120
  • Philanthropist has 40,320 possible spellings in TG!

    • This is what makes TG gibberish and not just a code. Each word doesn't have just one spelling, but many "synonyms"!
    If you want to make TG easier, don't scramble the word after you've removed the vowels.

    Rules/Example
    Take a word, "reading" for example. Remove all vowels: "Rdng". Now rearrange all of the letters except for the first and last ones: "Rndg".
    That is,

    1. Pick a word
    2. Remove vowels
    3. Scramble word but don't scramble the first and last letters. (Ex: Gbbrsh -> Bbgsrh wrong, Gbbrsh -> Gsbrbh correct)

    * At least I think this is correct...

    TI-BASIC Brainf**k Interpreter

    Here's an interpreter for Brainf**k, written in TI-BASIC, the programming language of the TI-84 graphing calculator: 

    
"   L₁ = tape
"   L₂ = stack
"    P = pointer/position
"    N = value of current cell
"    I = index of current character
" Str1 = BF code
" Str2 = current character
" Str3 = user input
" Str4 = output

ClrHome
201→dim(L₁)
Fill(0,L₁)
0→dim(L₂)
101→P
":"→Str4
Input Str1

For(I,1,length(Str1))
  sub(Str1,I,1)→Str2
  L₁(P)→N

  If Str2="+"
    N+1→L₁(P)

  If Str2="-"
    N-1→L₁(P)

  If Str2="<" and P>1
    P-1→P

  If Str2=">" and P<201
    P+1→P

  If Str2="." and N≤39 and N>0
    Str4+sub("ABCDEFGHIJKLMNOPQRSTUVWXYZ ,.0123456789",N,1)→Str4

  If Str2=",":Then
    Input Str3
    inString("ABCDEFGHIJKLMNOPQRSTUVWXYZ ,.0123456789",Str3)→L₁(P)
  End

  If Str2="["
    I→L₂(1+dim(L₂))

  If Str2="]":Then
    If N=0
      Then:dim(L₂)-1→dim(L₂)
      Else:L₂(dim(L₂))→I
    End
  End
End
